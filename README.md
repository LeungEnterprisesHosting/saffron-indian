# Saffron
Boilerplate for making a static site using Gulp.

## Instructions
`git clone https://github.com/LeungEnterprises/gulp-boilerplate`

`cd gulp-boilerplate`

`npm install`

`npm run bower:install`

`npm run bower:install -- wow --save`

`npm run gulp:dev`

`npm run gulp:build`

`npm run gulp:produce`
