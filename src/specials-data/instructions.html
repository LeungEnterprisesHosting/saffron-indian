How to Update the Specials

  1. Log into eNom.
  2. Once you're logged in, scroll down to "Core Tools" and click "Web Hosting".
  3. Then, click "Control Panel".
  4. Next, click "File Manager".
  5. In the File Manager, click the httpdocs folder.
  6. Then, click the specials-data folder.
  7. Next, click the folder of the current year (for example, 2017 if it's January 2017) and click the checkbox next to the current month (for example january.json)
  8. Then, click "Copy/Move" and change the destination to the next month (for example if it's January 2017): /httpdocs/specials-data/2017/february.json
  9. If it's December, you may need to create a new folder for the upcoming year (in the same place where the 2016 folder is).
  10. Next, edit the new file that was copied (february.json in this example) by clicking the pencil icon in the middle/
  11. Once you're done editing, ensure that it's in the correct format by copying the file contents into JSONLint.com and fixing any errors.
  12. If there are no errors, click save, then go back up to the original specials-data folder.
  13. Then, edit the current.json file with the new year and month. (Make sure to change the year if it's December!)
  14. Finally, make sure everything is OK by visiting the Specials page: Specials. Please allow up to an hour for it to update fully.
  15. If you have any questions, please feel free to email me! nathan@leungenterprises.com.
